# Taj Sons Website Guide

## Kaise Chalayen (How to Run)

1. **Server Start Karein**:
   Folder mein `server.js` file hai. Terminal mein likhein:
   ```bash
   npm start
   ```

2. **Website Kholein**:
   Browser mein yeh link kholein: [http://localhost:3000](http://localhost:3000)

3. **Admin Panel**:
   Products manage karne ke liye: [http://localhost:3000/admin](http://localhost:3000/admin)

## Configuration
`config.js` file ko edit karein aur apni details dalein:
- **geminiApiKey**: AI chat ke liye (Zaroori hai)
- **whatsappNumber**: Orders ke liye

## Products Add Karna
1. `Products` folder mein images dalein.
2. Admin panel par jayein aur "Scan New Products" buttons dabayein. 
3. Website par products automatic aa jayenge!

## Online Deploy Karna
Agar aap isay online (poori duniya ke liye) chalana chahte hain, toh Vercel ya Render use kar sakte hain.
